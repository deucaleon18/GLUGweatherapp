# GLUGweatherapp
This is my first project on GitHub which was a part of the audition process to the GNU and Linux users club of college,NIT Durgapur
I have used basic HTML,CSS and a lit bit of Javascript to create this project
I used the api present on the site https://api.openweathermap.org/data/2.5/weather for free to link the data of weather of various cities 
We need to enter the city to get the current weather statistics of the city.
